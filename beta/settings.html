Step 1: Environment Setup

  1. Create the Project:
    • Create a project folder:

      mkdir checkinpoint-app
cd checkinpoint-app
npm init -y

    • Install web3.js:

      npm install web3

  2. Configure MetaMask:
    • Add the Sepolia Base network (see Prerequisites).
    • Securely store your private key or account address.
  3. Prepare the ABI and Contract Address:
    • Save the ABI in a file named CheckInPointABI.json.
    • Sample simplified ABI:

      [
  {"constant": false, "inputs": [], "name": "checkIn", "outputs": [], "type": "function"},
  {"constant": false, "inputs": [{"name": "referralId", "type": "string"}], "name": "setReferralId", "outputs": [], "type": "function"}
]

    • Define the contract address:

      const contractAddress = '0xYourContractAddress';

  4. Create the Script: Create a file named index.js.
