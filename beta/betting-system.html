Step 5: Using the Betting Feature

Place one bet per day using a number and multiplier.

  1. Place a Bet:
    Call the placeBet function.

    async function placeBet(number, times, payoutAmount) {
  try {
    const result = await contract.methods.placeBet(number, times, payoutAmount).send({ from: userAddress });
    console.log('Bet placed successfully:', result);
    await checkBalance();
  } catch (error) {
    console.error('Error placing bet:', error.message);
  }
}
placeBet('01234', 2, 3);

  2. View Betting History:
    Check the user’s betting history.

    async function checkBetHistory() {
  try {
    const userBetCount = await contract.methods.getUserBetCount(userAddress).call();
    console.log(`Total Bets Made: ${userBetCount}`);
    if (userBetCount > 0) {
      const lastBet = await contract.methods.getUserBet(userAddress, userBetCount - 1).call();
      console.log('Most Recent Bet:', lastBet);
    }
    const totalBets = await contract.methods.getBetHistoryLength().call();
    console.log(`Total Global Bets: ${totalBets}`);
  } catch (error) {
    console.error('Error retrieving betting history:', error);
  }
}
checkBetHistory();

  3. Notes:
    • Parameters: Number (00–9999 characters), multiplier (1–10), payout amount (minimum 1).
    • Only one bet can be placed per day.
