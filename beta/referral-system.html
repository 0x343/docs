Step 4: Using the Referral Feature

Set your referral ID and earn 5 Points when a referee places a bet.

  1. Set Your Referral ID:
    Create a unique referral ID.

    async function setReferralId(referralId) {
  try {
    const result = await contract.methods.setReferralId(referralId).send({ from: userAddress });
    console.log(`Referral ID "${referralId}" has been set:`, result);
  } catch (error) {
    console.error('Failed to set referral ID:', error.message);
  }
}
setReferralId('myid123');

  2. Link a Referee:
    Connect another user to your referral.

    async function joinReferral(referralId, refereeAddress) {
  try {
    const result = await contract.methods.setReferralById(referralId).send({ from: refereeAddress });
    console.log(`Joined referral "${referralId}":`, result);
  } catch (error) {
    console.error('Failed to join referral:', error.message);
  }
}
joinReferral('myid123', '0xRefereeAddress');

  3. Check Referees:
    View your list of referees.

    async function checkReferees(referrerAddress) {
  try {
    const count = await contract.methods.getRefereeCount(referrerAddress).call();
    console.log(`Total Referees: ${count}`);
    if (count > 0) {
      const referees = await contract.methods.getReferees(referrerAddress).call();
      console.log('Referee List:', referees);
    }
  } catch (error) {
    console.error('Failed to retrieve referees:', error);
  }
}
checkReferees(userAddress);

  4. Note: The referral ID is unique and can only be set once. Rewards are granted when the referee places a bet.
