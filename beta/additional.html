Step 8: Utilizing Additional Functions

  1. Check Total Supply:
    Retrieve the total token supply.

    async function checkTotalSupply() {
  const supply = await contract.methods.totalSupply().call();
  console.log(`Total Supply: ${web3.utils.fromWei(supply, 'ether')} Points`);
}
checkTotalSupply();

  2. Monitor Events:
    Listen for the CheckIn event.

    contract.events.CheckIn({ filter: { user: userAddress } })
  .on('data', event => console.log('CheckIn Event:', event.returnValues))
  .on('error', error => console.error('Error:', error));

  3. Tips:
    • Never store your private key in the frontend code.
    • Always test your contract on the Sepolia testnet before deploying to mainnet.
    • Consider using ethers.js as an alternative to web3.js.
