Beta Testnet Documentation

(For Developers)


Introduction

HKBetaTestnet is a gamified smart contract deployed on the Sepolia Base Testnet network. This contract allows users to:

  • Daily Check-ins: Earn 1 Point per day by checking in.
  • Referral Program: Earn 5 Points when your referee places a bet.
  • Betting: Place bets with potential rewards based on selected numbers.

The Point token uses 18 decimals and direct transfers are disabled. This tutorial guides developers through interacting with the contract using web3.js, from initial setup to frontend integration.

Prerequisites

Please make sure you have the following:

  1. Crypto Wallet:
    • Install MetaMask.
    • Add the Sepolia Base Testnet network:

      Parameter Value
      RPC URL https://sepolia.base.org
      Chain ID 84532
      Symbol ETH

    • Claim Sepolia Base Testnet ETH from the:

      I. Alchemy: https://alchemy.com/faucets

      II. Chainlink: https://faucets.chain.link/base-sepolia

      III. L2faucet: https://l2faucet.com/base

      IV. Getblock: https://getblock.io/faucet/base-sepolia/

    • Paste the address, complete any required verification, and submit the request.
    • Wait for the ETH to arrive (typically 0.1–0.01 ETH).

      • Or Bridge from Sepolia to Sepolia Base Testnet on:

      Super Bridge: https://superbridge.app/base-sepolia


  2. Node.js and npm: Install Node.js (version 16 or above) and npm.
  3. web3.js: Install the library using the following command:
    Run this in your project terminal:

    npm install web3

  4. ABI: Get the ABI JSON file from the contract developer.
  5. Address: The address of the deployed contract on Sepolia Base Testnet (e.g., YourAddress).
  6. Text Editor: Use Visual Studio Code (VS Code) or any other code editor you prefer.
